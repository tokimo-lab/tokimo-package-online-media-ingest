# tokimo-package-online-media-ingest

Online media ingest service — download, transcode, and manage media from YouTube, Bilibili, and other platforms.

## Features

- yt-dlp based media download with format selection
- Audio extraction and post-processing
- Subtitle download and embedding
- Thumbnail generation
- Task queue management

## Add to Cargo.toml

```toml
tokimo-package-online-media-ingest = { git = "https://github.com/tokimo-lab/tokimo-package-online-media-ingest" }
```
